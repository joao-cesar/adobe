This panel simply lists the items from an FLA's Library.

The SWF and the JSFL must be in the "WindowSWF" folder under the user configuration folder.

Example:
C:\Users\joaoc\AppData\Local\Adobe\Animate 2020\en_US\Configuration\WindowSWF
